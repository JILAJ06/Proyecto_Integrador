Introducción

Este sistema fue creado para ayudar a las tiendas de abarrotes a controlar mejor sus ventas, inventario y productos. Permite registrar ventas y enviar tickets digitales, ver el historial, consultar las gráficas estadísticas de productos más vendidos y las ventas por semana. También guarda la información de empleados y clientes, y facilita el corte de caja al final del día.
